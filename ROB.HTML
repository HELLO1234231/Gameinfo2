GAME REVIEW

ROBLOX

Roblox is an online game platform and game creation system developed by Roblox Corporation. It allows users to program games and play games created by other users. Initial release date: 1 September 2006. It is a multiplayer game and the in game money is robux. And it was actually dynablocks. You can even make your own games and it even has it's own coding language called Lua.
